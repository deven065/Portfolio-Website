## 🎯 Animation Features

### Performance
- ⚡ 60 FPS on modern devices
- 🚀 GPU-accelerated transforms
- 🎨 Proper will-change properties
- 🔄 Efficient ScrollTrigger usage
- 🧹 Automatic cleanup with useGSAP

### Techniques Used
- **3D Transforms**: rotationX, rotationY, perspective
- **Easing**: power2, power3, back, sine, elastic
- **ScrollTrigger**: Viewport-based animations
- **Stagger**: Sequential delays
- **Timeline**: Coordinated sequences
- **Magnetic Effects**: Mouse-following interactions

### Timing
- Fast: 0.3s - 0.6s (UI interactions)
- Medium: 0.8s - 1s (reveals)
- Slow: 1.5s - 2s (counters, complex animations)
- Stagger: 0.05s - 0.2s delays
